Increasing Number of Digits after Decimal Point

Procedure to increase the number of digits after a decimal point.

You can increase the number of digits displayed after the decimal point, when displaying a floating point number.

The Panecal Scientific calculator can display a floating point number for upto 12 decimal places. Use this procedure to increase the number of digits displayed after the decimal point.

  1. Tap [ALT][FSE].
  2. Select FloatPt.
