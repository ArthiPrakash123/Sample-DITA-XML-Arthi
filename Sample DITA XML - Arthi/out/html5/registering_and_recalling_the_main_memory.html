Registering with and Recalling from the main memory

Demonstrates adding values to and recalling them from the main memory.

Use this procedure to store values in the main memory and then recall them when performing successive mathematical operations.

  1. Long press the MC button to clear the main memory.
    Important: Ensure that you always clear the main memory before you perform any mathematical operations to ensure correct results.
  2. Type a mathematical expression that uses mathematical operators, and click = to view the result.
  3. Long press the M+ button to add the result to the main memory.
  4. Type another mathematical expression and click = to view the result.
  5. Long press either the M+ or the M- button to add/subtract the result respectively in the main memory.
  6. Repeat steps 2 to 5 as necessary.
  7. Click ALT + M to recall the final result from the main memory.
